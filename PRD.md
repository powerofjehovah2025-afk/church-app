# Church Management App - PRD

## Project Vision
A unified web and mobile-friendly application to streamline church operations, volunteer management, and newcomer follow-up.

## Core Features
1. **Rota Management System**
   - Automated email/SMS notifications 14 days and 2 days before a shift.
   - Status tracking (Pending, Confirmed, Declined).
2. **Newcomer Pipeline**
   - Digital registration form.
   - Admin dashboard for tracking follow-up status.
3. **Live Church Calendar**
   - Unified view of all church events.
   - Logic for "Requesting" an event update.
4. **Admin Dashboard**
   - Secure area for department heads to manage their teams and data.

## Technical Stack
- Framework: Next.js (App Router)
- Database/Auth: Supabase
- UI: Tailwind CSS + Shadcn UI
- Notifications: Resend (Email) / Twilio (SMS/WhatsApp)
